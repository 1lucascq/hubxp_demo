<div className="sb-container">
  <h2>Componente ProductForm</h2>
  <p>
    O componente <code>ProductForm</code> é utilizado para criar ou editar um produto. Ele inclui campos para o <b>nome do produto</b>, <b>descrição</b>, <b>preço</b>, <b>upload de imagem</b> e <b>seleção de categorias</b>. O formulário pode ser pré-preenchido com dados iniciais para fins de edição.
  </p>
  <p>
    <b>Formas de utilização:</b> O componente pode ser utilizado tanto para criação quanto para edição de produtos. Para edição, é necessário passar a prop <code>initialData</code> com os dados do produto a ser editado.
  </p>
  <p>
    <b>Inputs:</b>
    <ul>
      <li><code>name</code>: Nome do produto (string)</li>
      <li><code>description</code>: Descrição do produto (string)</li>
      <li><code>price</code>: Preço do produto (number)</li>
      <li><code>imageFile</code>: Arquivo de imagem do produto (File)</li>
      <li><code>categories</code>: Categorias do produto (array de strings)</li>
    </ul>
	OBS: Todos os campos são obrigatório para criação, opcional para edição.
  </p>
  <p>
    <b>Envio da requisição:</b> A requisição é enviada utilizando um hook customizável que utiliza ReactQuery. O hook <code>useCreateProduct</code> é responsável por criar o produto, enquanto o <code>useEditProduct</code> é utilizado para editar um produto existente. Ambos os hooks fazem o cache dos resultados e invalidam as queries relevantes após a mutação.
  </p>

  <h2>Componente Dashboard</h2>
  <p>
    O componente <code>Dashboard</code> exibe indicadores chave de desempenho (KPIs) como total de pedidos, valor médio dos pedidos e receita total. Ele permite filtrar por categoria e produto para visualizar dados específicos.
  </p>
  <p>
    <b>Dados retornados:</b> O componente retorna os seguintes dados:
    <ul>
      <li><code>totalOrders</code>: Total de pedidos (number).</li>
      <li><code>averageOrderValue</code>: Valor médio dos pedidos (number).</li>
      <li><code>totalRevenue</code>: Receita total (number).</li>
    </ul>
  </p>
  <p>
    <b>Opções de filtro:</b> As opções de filtro incluem:
    <ul>
      <li><code>categoryId</code>: Filtrar por categoria.</li>
      <li><code>productId</code>: Filtrar por produto.</li>
    </ul>
  </p>
  <p>
    <b>Carregamento das opções de filtro:</b> As opções de filtro são carregadas através do estado das requisições feitas pelo ReactQuery no arquivo <code>App.tsx</code>.
  </p>
  <p>
    <b>Hook useDashboard:</b> O hook <code>useFetchDashboardData</code> é responsável por buscar os dados do dashboard. Ele faz o cache dos resultados pesquisados, evitando requisições desnecessárias. O hook utiliza o ReactQuery para gerenciar o estado das requisições e o cache dos dados.
  </p>
</div>
